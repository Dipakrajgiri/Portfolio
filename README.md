# Dipak Raj Giri - Personal Portfolio

Welcome to the source code for my personal portfolio! This is a responsive, fully-featured React web application highlighting my experience, projects, technical skills, and research background.

## About the Project
This portfolio serves as my digital resume and features a fast, modern design. 
It includes:
- **Dedicated Section Pages**: A full breakdown of my skills, projects, and work experience.
- **Dynamic Projects Showcase**: Interactive project cards covering domains like Artificial Intelligence, DevOps, and Full-Stack Development.
- **Responsive Layout**: Designed to look clean and professional on mobile phones, tablets, and desktop monitors.

## Built With
- **React.js & Vite** for lightning-fast builds and rendering.
- **TypeScript** for robust type-safety.
- **Tailwind CSS v4** for clean, utility-first styling.
- **Framer Motion** for smooth transition animations.
- **GitHub Actions** for an automated CI/CD deployment pipeline.

## Hosting
The site is hosted automatically using GitHub Pages from the `main` branch.
